# WPT Alignment Prototype BOM

หน้าเว็บนี้เป็นรายงาน `Bill of Materials` สำหรับต้นแบบ `Wireless Power Transfer Alignment Prototype` ขนาด `36W`

## เวอร์ชันที่มีใน repo

- `index.html` เวอร์ชันต้นฉบับ ใช้เป็นหน้าแรกของ GitHub Pages
- `original-report.html` สำเนาไฟล์ต้นฉบับเดิม
- `premium-report.html` เวอร์ชันที่ปรับดีไซน์ใหม่

## เปิดดูในเครื่อง

เปิดไฟล์ [index.html](/Users/User/Documents/New project/index.html) ด้วยเบราว์เซอร์ได้ทันที

## เปิดผ่าน GitHub Pages

1. push ไฟล์ในโฟลเดอร์นี้ขึ้น GitHub
2. ไปที่ `Settings > Pages`
3. ตั้งค่า `Build and deployment` เป็น `Deploy from a branch`
4. เลือก branch `master` และโฟลเดอร์ `/ (root)`
5. บันทึก แล้วรอ GitHub Pages publish

เมื่อ publish เสร็จ หน้าเว็บหลักจะใช้ `index.html` ซึ่งเป็นเวอร์ชันต้นฉบับ
