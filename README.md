# WPT Alignment Prototype BOM

หน้าเว็บนี้เป็นรายงาน `Bill of Materials` สำหรับต้นแบบ `Wireless Power Transfer Alignment Prototype` ขนาด `36W` และถูกจัดให้เปิดได้สะดวกบน GitHub หรือ GitHub Pages

## เปิดดูในเครื่อง

เปิดไฟล์ [index.html](/Users/User/Documents/New project/index.html) ด้วยเบราว์เซอร์ได้ทันที

## เปิดผ่าน GitHub Pages

1. สร้าง repository ใหม่บน GitHub
2. push ไฟล์ในโฟลเดอร์นี้ขึ้น branch `main`
3. ไปที่ `Settings > Pages`
4. ตั้งค่า `Build and deployment` เป็น `Deploy from a branch`
5. เลือก branch `main` และโฟลเดอร์ `/ (root)`
6. บันทึก แล้วรอ GitHub Pages publish

เมื่อ publish เสร็จ หน้าเว็บจะเปิดได้จาก URL ของ GitHub Pages โดยตรง

- `README.md` คำอธิบายและวิธีเปิดใช้งาน
